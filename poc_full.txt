1. PoC概要
本指示書は、Word SurvivorsのPoC（Proof of Concept）をGodot 4.xで実装するための技術仕様書です。Claude Codeでの開発を前提とし、各ファイルの役割・実装順序・具体的なコード仕様を記載します。

1.1 PoCスコープ
Stage 1のみ（単語プール200語）
武器4種（名詞2 + 動詞2）
武器進化2組（接尾辞 -tion, -er）
シナジー3組
段階的情報開示システム
基本図鑑
レベルアップシステム
Web export対応（itch.io公開前提）

4.5 exp_gem.gd
EXPジェムは敵が死亡時に生成。プレイヤーのPickupAreaに入ると吸寄せられて回収される。

# Area2Dベース
var exp_amount: int = 1
var is_being_collected: bool = false
var collect_speed: float = 400.0

func _physics_process(delta):
    if is_being_collected:
        var dir = (player_ref.global_position - global_position).normalized()
        global_position += dir * collect_speed * delta
        if global_position.distance_to(player_ref.global_position) < 10:
            player_ref.add_exp(exp_amount)
            queue_free()

func try_evolve(weapon_node) -> bool:
    var word = weapon_node.word_data
    if word.has("evolves_to") and word.has("evolution_passive"):
        if word.evolution_passive in player.owned_passives:
            if weapon_node.level >= 8:  # Lv8以上で進化可能

4.8 enemy_spawner.gd
敵のスポーンに SRS ロジックを組み込む。

# スポーンロジックの概要:
# 1. タイマーで定期的に敵をスポーン
# 2. WordDatabase.get_random_word() で単語を取得
#    - SRS重み付けで「久しぶりの単語」が出やすい
# 3. 品詞に応じた敵タイプを生成
# 4. 画面外のランダム位置に配置

var spawn_interval: float = 1.0  # 秒
var max_enemies: int = 80
var difficulty_timer: float = 0.0

func _on_spawn_timer_timeout():
    if get_tree().get_nodes_in_group("enemies").size() >= max_enemies:
        return
    var word = WordDatabase.get_random_word()
    var enemy = enemy_scene.instantiate()
    enemy.setup(word)
    enemy.global_position = get_spawn_position()
    add_child(enemy)

func get_spawn_position() -> Vector2:
    # プレイヤーから半径400〜600の円上のランダム位置
    var angle = randf() * TAU
    var dist = randf_range(400, 600)
    return player.global_position + Vector2.from_angle(angle) * dist

9. Claude Code向け開発ノート

9.1 ファイル作成の注意点
.tscnファイルはGodotのテキスト形式。手書き可能だが複雑なので、.gdファイルを先に書いてGodotエディタでシーンを組み立てるのが安全
Autoloadの登録はproject.godotに直接記述可能
JSONデータはClaude Codeで全生成可能。学習指導要領準拠の単語リストを元に200語を選定

9.2 実装優先度
① まずプレイヤーが動いて敵が来て死ぬ状態を作る
② 武器1種だけで敵が倒せる状態を作る
③ EXPジェムとレベルアップを入れる
④ 段階的情報開示を入れる
⑤ 武器2種目以降を追加
⑥ 進化・シナジーを入れる
⑦ UI（タイトル、図鑑、リザルト）を入れる

9.3 テスト基準
プレイヤーが15分生存できるバランス
同じ単語に5回以上遭遇42したらマスター判定が正しく動作
武器進化が正しく発動
シナジーボーナスが正しく適用
図鑑にマスター単語が正しく登録
Web exportでブラウザ動作確認
日本語フォントが正しく表示
